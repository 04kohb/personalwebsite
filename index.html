Surbana Jurong Internship Project

I created a simple lift monitoring device by writing a C program using Raspberry Pi and serial communication (UART) to transmit information between the Raspberry Pi and the laptop to send lift fault messages. This project is significant because most of the software and hardware of Lift Monitoring devices are developed my external companies. This project was created to help Surbana Jurong study lift monitoring devices so they can devep their own lift monitoring software in the future.
